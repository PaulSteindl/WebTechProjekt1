ERSTELLEN EINES USERS

Um sich in unserer Website zu registrieren befindet sich an der rechten oberen Ecke ein Feld "Sign up". Klicken Sie dieses, um auf ein Formular weitergeleitet zu werden, um Ihre persoenlichen Daten einzugeben. Wenn dies erledigt, anschließend auf "Sign up" druecken und schon sind Sie angemeldet. Dies erkennen Sie an Ihrem angezeigten Username in der rechten oberen Ecke.

ANMELDEN EINES USERS

Zum Anmelden bitte das Feld "Login" im rechten oberen Bereich auswaehlen und dort Ihre Benutzerdaten eingeben. Anschließend auf "Login" und Sie sind angemeldet.

PASSWORT VERGESSEN

Falls Sie sich im Login Bereich befinden und Sie Ihr Passwort nicht mehr wissen, gibt es die Moeglichkeit dieses mithilfe Ihrer E-Mail-Adresse oder Username wieder zurueckzusetzten. Dafuer das Feld "Forgot your Password" waehlen. Sie werden auf eine Seite weitergeleitet, wo Sie dieses abaendern koennen. Das alte Passwort darf nicht dasselbe sein wie das neue Passwort.

Ihr Password muss saemtliche Faelle erfuellen:

Mindestens einen Großbuchstaben

Mindestens acht Zeichen lang sein

Mindestens einen Kleinbuchstaben

KONTAKT

Ihnen wird im unteren Bereich das Impressum angezeigt. Dies fuehrt Sie zu unseren Kontakten, falls Sie uns kontaktieren moechten.

BEITRaeGE ERSTELLEN

Um einen Beitrag zu erstellen gibt es das Feld "New Post". Dieses befindet sich oben in der Navigation-Bar. Sie koennen je nach Wunsch einen reinen Textbeitrag erstellen. Weiters koennen Sie aber auch Bilder als auch saemtliche tags hinzufuegen. Wahlweise gibt es auch die Moeglichkeit den Beitrag in verschiedenen Formen zu veroeffentlichen:

oeffentlich = fuer Gaeste-User als auch Registrierte-User.

Beschraenkt = nur fuer Registrierte-User.

Privat = nur fuer sich selbst sichtbar.

Anschließen auf "Beitrag erstellen".

BEITRaeGE ANSEHEN

In der Navigation-Bar befindet sich auch das Feld "Posts". Dort koennen Sie alle Beitraege sehen die auf oeffentlich oder auch beschraenkt gestellt worden sind, sofern Sie kein Gaeste-User sind.

KOMMENTIEREN VON BEITRaeGEN

Befinden Sie sich in der Beitragsanzeige koennen sie auch einen gewuenschten Beitrag kommentieren. Unter einen Post werden alle von Usern veroeffentlichen Kommentare angezeigt. Sie koennen dort auch andere Profile ansehen, indem Sie auf deren Username klicken.

PROFIL ANZEIGEN

Um Ihr Profil anzeigen zu lassen, druecken Sie bei einem Ihrer Posts / Kommentare auf Ihren Username. Ihr Profilbild, Benutzername und weiteres wird hier angezeigt. Andere User sehen diese Daten auch.

REAGIEREN VON BEITRaeGEN

Um eine Reaktion auf eines anderen Users Beitrag zu hinterlassen, koennen Sie dies mittels eines Buttons. Liken / Disliken Sie diesen Post und zeigen Sie dem Ersteller das Ihnen der Post gefaellt / nicht gefaellt. Der Ersteller kann dies sehen.

PROFILE SETTINGS

Druecken Sie auf "Profile" kommen Sie in die Profile Settings. Dort koennen Sie auch saemtliche aenderungen zu Ihrem Profil vornehmen.

AUSLOGGEN

Loggen Sie sich mithilfe des "Logout" Buttons von Ihrem Profil aus, beenden Sie Ihre aktuelle Session. So verhindern Sie, dass Unbefugte in Ihr Postfach gelangen koennen, z.B. ueber die Zurueck-Funktion Ihres Internet-Browsers.
